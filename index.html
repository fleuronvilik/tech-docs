IF STATEMENTS

If statements allow you to run specific section of code when a test is true. The code inside the parentheses () is the test. If the test is true then the code inside the block {} runs. If the test is not true, the the code inside the block doesn't run.

          
            var aNumber = 5;
            if (aNumber === 5) {
                drawBox(blue);
            }
            if (aNumber === 4) {
                drawBox(red);
            }

LOOPS

  1. Use a for loop
  2. Used a Classic For Loop
  3. Used Nested for Loop

A loop repeats the section of code within curly brackets {}. In JavaScript, a for of loop goes through each element in an array or string repeating the code for each element.

          
            var groceries = [
              'apples',
              'yogurt',
              'toothpaste'
            ]
            for (var element of groceries) {
              print(element);
            }

USED A CLASSICAL FOR LOOP II

In Javascript, a classic for loop repeats the same code inside the block { } a specified number of times. It is defined in three parts. The first part happens before the for loop starts and defines a looping variable (var i = 0;). The second part is a test that determines if the loop should keep repeating (i < 10;). And the third part updates the looping variable each time the loop repeats (i = i + 1).

            
              print('The loop will go as long as i is less than 10');
              for (var i = 0; i < 10; i = i + 1) {
                  print(i);
              }
